Gets a leaderboard(s) (getLeaderboard)
Retrieves the set of leaderboards based on the reqest

Query parameters

Query Parameter — optional resetWindow to get a specific leaderboard. If not specified will return all windows

Query Parameter — optional game mode. If not specified all game modes will be retrieved

get /UserScore
Gets a users score type for one or more game types (getUserScore)
Gets a users score information

Query parameters

userId (optional)
Query Parameter — The userId to retrieve the score format: int64
gameMode (optional)
Query Parameter — optional game mode. If not specified all game modes will be retrieved

Return type

array[UserScoreData]

Content-Type: application/json
